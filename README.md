# Voting DApp

A **Decentralized Voting Application** built using **React.js, Solidity, and Hardhat** that allows users to vote securely on the blockchain.

## Features

- **Decentralized Voting System** using Ethereum blockchain
- **Smart Contract-based Voting** with Solidity
- **User-friendly React.js Interface**
- **Real-time Vote Counting**
- **Secure and Transparent** (Immutable results)
- **Metamask Integration** for authentication

## Technologies Used

- **React.js** – Front-end framework
- **Solidity** – Smart contract development
- **Hardhat** – Smart contract testing & deployment
- **Ethers.js** – Blockchain interaction
- **Metamask** – Wallet for Ethereum transactions
- **CSS** – Styling UI

## Installation

### Prerequisites
Ensure you have the following installed:
- **Node.js** & **npm**
- **Metamask Extension**
- **Hardhat** (for smart contract deployment)

## Usage

1. **Connect Wallet**: Click on "Connect Metamask".
2. **Register Candidates**: Admin adds candidates via smart contract.
3. **Vote**: Users vote for their preferred candidate.
4. **View Results**: Real-time updates on the voting results.
